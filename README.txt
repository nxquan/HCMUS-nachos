Prerequisite:
- Unix Operating System: Ubuntu 14.04 or CentOS 6 
- gcc 4.x

How to run nachos 3.4?
First, you must give executable files in folder source executable permission 
That executable files includes:
- gnu-decstation-ultrix / decstation-ultrix / 2.95.3 / *
- code/userprog/nachos

Second, open terminal and change directory to folder code
Third, input "make" in terminal and enter for compiling source code. Afterward, you can test a file c in folder test with command "./userprog/nachos -rs 1023 -x ./test/file_name"
